# FSC guide 2020 by Bahna

It's a [pollen](https://docs.racket-lang.org/pollen/) project with PDF and a website as output. 


To make a book, run:

```shell
$ make book
```

To make a website:

```shell
$ make site
```

To export all built files into the `docs` folder ready for serving by a file server:

```shell
$ make export
```
